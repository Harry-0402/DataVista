Getting Started

1
Download the universal manifest.xml file.
Download Manifest
2
Open Excel (Desktop or Web) and go to Insert > Add-ins > Manage My Add-ins.
3
Select Upload My Add-in and choose the manifest.xml file you downloaded.
4
Success! Use the new DataVista tab in the Ribbon to export your data.
Note: This Add-in runs entirely in your browser. No data is sent to any server.
Built with DataVista Engine v2.0 (Ribbon-First)
