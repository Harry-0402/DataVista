How to Install

1
Download the manifest.xml file.
Download Manifest
2
Open Excel (Desktop or Web).
3
Go to Insert > Add-ins > Manage My Add-ins.
4
Select Upload My Add-in and choose the manifest.
Note: Since this is hosted on GitHub Pages, it will work persistently without any local server!
